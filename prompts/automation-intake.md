## Usage

Used exclusively by Cursor Automation **CP — Intake** (see [docs/automations.md](../docs/automations.md)).

1. Open PR from branch `intake/{slug}` with `## Intake` section in the PR body.
2. Add label `cp:intake`.
3. Agent delegates to [intake-v1.md](intake-v1.md).
